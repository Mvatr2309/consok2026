## 1. Для абитуриентов (запись на консультацию)

### Как записаться

1. Перейдите по ссылке на нужную программу (ссылку предоставляет администратор или можно выбрать программу на главной странице).
2. Выберите эксперта из списка.
3. Выберите удобный слот (дата и время). Отображаются только слоты на ближайшие 5 дней, у которых есть свободные места.
4. Введите ваше **имя** и **email**.
5. Нажмите **«Записаться»**.
6. После успешной записи вы увидите:
   - Ссылку на встречу
   - Кнопки **«Добавить в Google Календарь»** и **«Добавить в Яндекс Календарь»**
7. На указанный email придёт письмо с подтверждением и деталями встречи.
8. За 1 час до встречи придёт письмо-напоминание со ссылкой на встречу (если включено администратором).

## 2. Для экспертов (личный кабинет)

### Вход в кабинет

1. Перейдите на https://miptconsultations.icust.online/expert
2. Введите **логин** и **пароль**, выданные администратором.
3. Нажмите **«Войти»**.

### Возможности кабинета

- **Предстоящие консультации** — список ваших будущих слотов с информацией о записавшихся (имя, email, дата записи). Отображается по 10 штук с пагинацией.
- **Архив** — прошедшие встречи с историей записей.
- **Ссылка на встречу** — отображается вверху кабинета.
- **Выход** — кнопка «Выйти» в правом верхнем углу.

## 3. Для администратора (админ-панель)

### Вход

1. Перейдите на https://miptconsultations.icust.online/admin
2. Введите логин и пароль HTTP Basic Auth (заданы в переменных окружения `ADMIN_USER` и `ADMIN_PASS`).

### Вкладки админ-панели

#### Программы
- Просмотр, создание, редактирование и удаление образовательных программ.
- Привязка экспертов к программам.
- Столбец **«Ссылка для абитуриентов»** — готовая ссылка на страницу записи, с кнопкой «Копировать».

#### Эксперты
- Просмотр, создание, редактирование и удаление экспертов.
- Загрузка фото эксперта.
- Указание ссылки на встречу (Google Meet / Zoom / др.).
- Привязка к программам.
- Задание **логина и пароля** для входа в личный кабинет (раздел «Авторизация в кабинет» в форме редактирования). При редактировании оставьте поле пароля пустым, чтобы не менять текущий.
- Столбец **«Логин»** — текущий логин эксперта.
- Столбец **«Кабинет эксперта»** — прямая ссылка на кабинет (по токену), с кнопкой «Копировать».

#### Слоты
- Просмотр всех слотов с информацией: дата/время, программа, эксперт, занятость мест, ссылка на встречу.
- **Создание** слотов вручную (кнопка «+ Добавить»).
- **Импорт из CSV** — массовое создание слотов:
  1. Нажмите «Скачать шаблон CSV» для получения шаблона.
  2. Заполните файл: `Эксперт;Программа;Дата и время (ГГГГ-ММ-ДД ЧЧ:ММ);Макс. участников`
  3. Нажмите «Загрузить CSV».
- **Массовое удаление**:
  1. Отметьте чекбоксами нужные слоты (или «Выделить все» в заголовке).
  2. Нажмите **«Удалить выбранные (N)»**.
  3. Подтвердите удаление. Связанные записи (букинги) также будут удалены.
- Редактирование и удаление отдельных слотов.

#### Записи
- Просмотр всех записей абитуриентов: имя, email, программа, эксперт, дата слота, дата записи.
- Удаление отдельных записей.

#### Уведомления
- **Письмо при регистрации** — включить/выключить автоматическую отправку письма-подтверждения при записи.
- **Напоминание перед встречей** — включить/выключить и настроить за сколько минут до встречи отправлять напоминание (по умолчанию 60 минут).
